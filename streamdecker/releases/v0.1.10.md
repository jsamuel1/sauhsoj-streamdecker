## Streamdecker v0.1.10

Stream Deck controller with Kiro profile support.

### Downloads

- **Streamdecker.dmg** - Full app (signed & notarized)
- **wtf.sauhsoj.streamdecker.streamDeckPlugin** - Elgato plugin only
- **Kiro-Neo.streamDeckProfile** - Neo profile (8 buttons)
- **Kiro-Mini.streamDeckProfile** - Mini profile (6 buttons)

### Changes

- Renamed app from "Kiro Deck" to "Streamdecker"
- Fixed icon paths after consolidation
- Added Mini profile for Stream Deck Mini
- Plugin and profiles now available as separate downloads
- Config directory moved to `~/.config/streamdecker/`

### Features

- 8 buttons (Neo) / 6 buttons (Mini) for kiro-cli control
- Focus, Cycle, Alert, Launch, Yes, No, Trust, Agent actions
- Info bar status display (Neo only)
- Web emulator at http://127.0.0.1:3848
- Elgato mode with plugin/profile management

### Installation

**Full App:**
1. Download `Streamdecker.dmg`
2. Drag `Streamdecker.app` to `/Applications`
3. Launch from Applications

**Plugin Only:**
1. Download `wtf.sauhsoj.streamdecker.streamDeckPlugin`
2. Double-click to install
3. Download and import the profile for your device

### Requirements

- macOS 12+
- Stream Deck Neo or Mini
- iTerm2 (for kiro-cli integration)
