## Kiro Deck v0.1.5

Plugin installer fix.

### Fixes

- Fixed .streamDeckPlugin zip structure - now includes the .sdPlugin folder correctly
- Elgato Stream Deck can now install the plugin

### Installation

1. Download `Kiro Deck.dmg`
2. Drag `Kiro Deck.app` to `/Applications`
3. Launch from Applications or Spotlight
