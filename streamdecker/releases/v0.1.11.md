## Streamdecker v0.1.11

### New Features

- **Auto-Update** - Check for updates from GitHub releases
  - Silent check on startup with notification if update available
  - "Check for Updates..." menu item in menubar
  - One-click download and install

### Changes

- Menubar now shows version number (e.g., "Streamdecker v0.1.11")
- Tooltip updated to "Streamdecker"

### Downloads

- **Streamdecker.dmg** - Full app (signed & notarized)
- **wtf.sauhsoj.streamdecker.streamDeckPlugin** - Elgato plugin only
- **Kiro-Neo.streamDeckProfile** - Neo profile (8 buttons)
- **Kiro-Mini.streamDeckProfile** - Mini profile (6 buttons)

### Installation

**Full App:**
1. Download `Streamdecker.dmg`
2. Drag `Streamdecker.app` to `/Applications`
3. Launch from Applications

**Plugin Only:**
1. Download `wtf.sauhsoj.streamdecker.streamDeckPlugin`
2. Double-click to install
3. Download and import the profile for your device
