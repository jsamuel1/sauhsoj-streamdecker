## Kiro Deck v0.1.4

Emulator preview fix.

### Fixes

- Button icons now render in emulator preview regardless of mode
- Previously icons only showed when in standalone mode with device connected

### Installation

1. Download `Kiro Deck.dmg`
2. Drag `Kiro Deck.app` to `/Applications`
3. Launch from Applications or Spotlight
