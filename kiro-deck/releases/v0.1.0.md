## Kiro Deck v0.1.0

Stream Deck controller for kiro-cli.

### Features
- 8 buttons (Neo) / 6 buttons (Mini) for kiro-cli control
- Focus, Cycle, Alert, Launch, Yes, No, Trust, Agent actions
- Agent picker page - select agents directly from Stream Deck
- Info bar status display with agent switching feedback
- Web emulator at http://127.0.0.1:3848
- Menubar tray with native config window
- Keyboard shortcuts for agent switching (configurable)
- fzf-based project folder picker for Launch

### Installation
1. Download `Kiro Deck.zip`
2. Extract and move `Kiro Deck.app` to `/Applications`
3. Launch from Applications or Spotlight

### Requirements
- macOS 12+
- Stream Deck Neo or Mini
- iTerm2
- kiro-cli
