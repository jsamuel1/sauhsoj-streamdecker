## Kiro Deck v0.1.3

Bug fixes and mode switching improvements.

### Fixes

- Fixed app not launching after install (shared modules path issue)
- Stream Deck connection now only active in standalone mode
- Mode switch properly disconnects/reconnects to device

### Mode Switching Improvements

- **elgato → standalone**: Option to remove Elgato from login items
- **standalone → elgato**: Option to add Elgato to login items
- Checkbox shown in mode switch dialog (checked by default)

### Structure Changes

- Moved `shared/` into `kiro-deck/` for correct relative imports in app bundle

### Installation

1. Download `Kiro Deck.dmg`
2. Drag `Kiro Deck.app` to `/Applications`
3. Launch from Applications or Spotlight

### Requirements

- macOS 12+
- Stream Deck Neo or Mini
- iTerm2

### Signed & Notarized

This release is code-signed and notarized by Apple.
